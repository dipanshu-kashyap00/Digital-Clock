# Digital Clock (C)

A console-based digital clock application written in C that displays the current system time and date in real time. The program supports both 24-hour and 12-hour (AM/PM) formats and updates automatically every second.

## Features

- Real-time clock display
- 24-hour format
- 12-hour format (AM/PM)
- Displays current date
- Continuous automatic refresh

## Technologies

- C Programming Language
- Standard C Library (`time.h`, `stdio.h`, `stdlib.h`)

## Build and Run

### Linux / macOS
```
gcc DigitalClock.c -o DigitalClock
./DigitalClock
```

### Windows
```
gcc DigitalClock.c -o DigitalClock.exe
DigitalClock.exe
```
